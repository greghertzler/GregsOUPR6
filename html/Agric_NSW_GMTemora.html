Agric_NSW_GMTemoraR Documentation

Farm Adaptation at Temora, New South Wales

Description

APSIM generated gross margins for wheat (W), canola (C) and sheep (S)

Format

csv file with 50 rows and 5 columns

Details

  • Year: time variable in annual increments

  • CWWCWW GM: Gross margins per hectare for CWWCWW rotation

  • CWWSSS GM: Gross margins per hectare for CWWSSS rotation

  • SSSSC GM: Gross margins per hectare for SSSSC rotation

  • S GM: Gross margins per hectare for continuous S

Source

https://nccarf.edu.au/will-primary-producers-continue-adjust-practices-and-technologies-change-production/
