Climate_TempsMax_CapeOtwayR Documentation

Maximum daily temperatures at Cape Otway

Description

Bureau of Meteorology, raw (unhomogenized) data.

Format

csv file with 8888 rows and 2 columns

Details

  • Year: time variable in daily increments

  • Max: maximum temperature in degrees centigrade

Source

http://www.bom.gov.au/climate/data/?ref=ftr, Station number 090015
